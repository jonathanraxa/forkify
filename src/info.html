Command Line Basics

  • ls | dir - see the contents of the folder
  • cd - change directory
  • clear - clears the command line
  • mkdir *nameOfFolder* - creates a directory
  • touch *nameoffile.html* | copy NUL *nameoffile.html* - creates a file with the given name
  • cp | copy - 'cp jonas.js .. ' --> copies 'jonas.js' into the parent folder
  • mv | move - 'mv test.js ..' --> moves the file
  • rm | del - 'rm test.js ..' --> removes the file entirely
  • rm -r | rmdir (the entire folder) - 'rm -r test.js ..' --> removes the folder entirely (-r) means recursively
  • open *nameOfFile* - opens the file in the default program
  • install npm - installs npm
  • npm init - initializes the package with a package.json file
  • npm install webpack --save-dev - saves package as a dependency for the project
    • WEBPACK is a development tool
  • npm install jquery --save
  • devDependencies (npm install *** --save-dev) - modules which are only required during development
  • dependencies (npm install *** --save) - modules which are also required at runtime
  • notes: we only need to share the package.json file which list the dependencies that we have. "NPM INSTALL" will install all the dependencies that we need straight from the JSON file.
  • npm uninstall jquery --save - uninstalls the package
  • npm install live-server --global | -g - installs the package globally
